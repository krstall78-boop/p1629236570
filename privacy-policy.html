جمع البيانات Data Collection

في متجر كرستال بلس، نجمع البيانات الضرورية فقط لإتمام طلباتك وتحسين تجربتك، وتشمل: الاسم، رقم الجوال، البريد الإلكتروني، وعنوان الشحن. نحن لا نقوم بتخزين بيانات بطاقات الدفع الائتمانية إطلاقاً، حيث يتم معالجتها عبر بوابات دفع مشفرة وآمنة.

أمن البيانات Data Security

نستخدم بروتوكولات حماية متطورة (SSL/HTTPS) لتشفير كافة البيانات المتبادلة. لا نقوم ببيع أو تأجير بياناتكم لأي أطراف ثالثة، ويتم مشاركة البيانات الضرورية فقط مع شركات الشحن لإيصال المنتج إليكم.
